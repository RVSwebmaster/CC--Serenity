# Serenity Greenhorn Builder Screen Flow

## Goal
Guide a player through revised Greenhorn creation and end on a printable digital sheet.

## Steps
1. Basics
   - name
   - quote
   - concept
   - role on the crew
   - homeworld
   - short background
2. Attributes
   - six Attributes
   - live point tracking
   - derived Life Points and Initiative
3. Traits
   - Assets and Complications
   - revised Trait Point economy
   - at least one of each required
4. Skills
   - general skills and specialties
   - live point tracking
   - enforce d6 general cap and d8 specialty minimum
5. Details
   - gear
   - notes
   - optional portrait URL
6. Character Sheet
   - validation summary
   - full read-only digital sheet
   - print from browser

## MVP constraints
- no backend
- no login
- no user accounts
- localStorage save only
- JSON export and import
- original presentation, not a reproduction of an official sheet
